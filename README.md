# Proiect T1 – Audit Linux

## Structura directoare

- bin/ – fișiere executabile
- src/ – cod sursă
- include/ – fișiere header
- data/ – fișiere de date utilizate în proiect
- logs/ – fișiere jurnal
- reports/ – rapoarte generate de scriptul de audit
- tmp/ – fișiere temporare
- tests/ – fișiere pentru testare
- doc/ – documentația proiectului
- tools/ – scripturi utilitare (ex: audit)
